# Ticket+ Movie Browsing Platform

A responsive movie browsing web application with search functionality and a catalog of 3000+ movies.

**Live Demo:** [movie-site-smoky.vercel.app](https://movie-site-smoky.vercel.app)

## About

Ticket+ allows users to search and browse movies with details including ratings, runtime, and language. Built with HTML5, CSS3, and JavaScript with API integration for movie data.

## Features

- Real-time movie search
- Responsive design for all devices
- Movie details (posters, ratings, runtime, language)
- Clean, modern interface

## Technologies

- HTML5, CSS3, JavaScript (ES6+)
- API Integration
- Deployed on Vercel
